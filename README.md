# Simple Calculator (CLI)

A simple Command Line Interface (CLI) calculator built using Python. This program takes two numbers as input from the user and performs basic arithmetic operations.

# Features

- Addition
- Subtraction
- Multiplication
- Division

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Angad1955/synent-task1-simplecalculator-angad
   ```

2. Navigate to the project directory:
   ```bash
   cd synent-task1-simplecalculator-angad
   ```

3. Run the program:
   ```bash
   python task_1.py
   ```

# Example Output

```
Simple Calculator:
Enter First number: 12
Enter Second number: 4

12 + 4 = 16
12 - 4 = 8
12 * 4 = 48
12 / 4 = 3.0
```

If the second number is zero:

```
Simple Calculator:
Enter First number: 10
Enter Second number: 0

10 + 0 = 10
10 - 0 = 10
10 * 0 = 0
Cannot divide by zero!
```
